# 蘇幕遮 年三十食於海上皇宮
20150221

倚東船，望南浦，西殿如林，北雁圍樑柱。

畫舸輕煙穿萬縷，廣袖長襟，舞落千家雨。

發春華，凋秋樹，冬雪為嵐，夏雀鳴檐宇。

覓得佳人傾肺腑，攬勝尋芳，去日知何數。
