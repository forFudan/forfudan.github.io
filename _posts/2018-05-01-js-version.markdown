# 椴树蜜 / Linde Honing / Miel de tilleul 

外观：乳白色，微黄色，凝脂状。    
气味：柔和，微酸。    
味道：爽口，清凉，有薄荷感，微甜，不腻。    
评价：气味纯正，口感清爽，甜度适中，不冲不腻。    
推荐：五星，推荐。
