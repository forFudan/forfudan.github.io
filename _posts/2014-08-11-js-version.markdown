# 詠牛肉包子
20140811

大南門牛肉包子。俗稱安慶美食第一，噴香可口，回味無窮。據說每天早上六七點早餐時間，排隊排到人山人海，一路堆滿盛唐山。為詩曰：

晨興塔影對江彎，一水遙接鎮海關。

莫問佳餚何處覓，行人盡望盛唐山。
